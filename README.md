# Fintrack Project
A Web app Project that is for tracking and managing budgets

## Overview
This is a web app, for tracking finance. It uses the concept of splitting money into budgets for managing, so that the money can be monitored as 'purposes'. 
This project aims to automate finance tracking and visualize the budget flow, for people who are interested in seeing where their money goes to (including me).
